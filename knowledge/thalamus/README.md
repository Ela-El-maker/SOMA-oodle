# THALAMUS — Security/Risk Lobe Knowledge Library

**Domain:** Security incidents, risk events, policy decisions, threat modeling, anomaly patterns

This folder is THALAMUS's training dataset. Fine-tuning here makes SOMA's risk
lobe genuinely paranoid about *known* failure modes — it learns what has actually
gone wrong in this system, not generic security advice.

## What belongs here
- Security incidents (API key leaks, auth failures, permission issues)
- Anomalies detected and their resolution
- Policy decisions (what was allowed/blocked and why)
- Health warnings with context about what caused them
- Risk tradeoffs that were explicitly made

## Entry format (auto-generated by KnowledgeCuratorArbiter)
```
---
lobe: thalamus
type: [security_incident | anomaly | policy_decision | risk_tradeoff | health_event]
source: [signal type that triggered this]
timestamp: ISO-8601
severity: [low | medium | high | critical]
resolved: true|false
---
[content]
```

## Training threshold
KnowledgeCuratorArbiter will emit `training.threshold.ready` when this folder
reaches **100 entries**. OllamaAutoTrainer will propose a THALAMUS LoRA fine-tune
to Barry for human-gated promotion.
