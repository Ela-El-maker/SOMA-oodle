# LOGOS — Rationalist Lobe Knowledge Library

**Domain:** Logic, engineering, code, architecture, debugging, systems design

This folder is LOGOS's training dataset. Every entry here will eventually fine-tune
the LOGOS LoRA adapter — making SOMA's engineering brain genuinely smarter about
*SOMA-specific* patterns, not just generic code.

## What belongs here
- Successful refactors and the reasoning behind them
- Architecture decisions (what we chose and why)
- Bug post-mortems (what failed, root cause, fix)
- Signal flow descriptions, API contracts, wiring patterns
- Code patterns that are idiomatic to SOMA's codebase

## Entry format (auto-generated by KnowledgeCuratorArbiter)
```
---
lobe: logos
type: [swarm_outcome | architecture_decision | bug_postmortem | insight]
source: [signal type that triggered this]
timestamp: ISO-8601
confidence: 0.0-1.0
---
[content]
```

## Training threshold
KnowledgeCuratorArbiter will emit `training.threshold.ready` when this folder
reaches **100 entries**. OllamaAutoTrainer will propose a LOGOS LoRA fine-tune
to Barry for human-gated promotion.
