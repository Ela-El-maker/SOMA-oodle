# AURORA — Creative/Soul Lobe Knowledge Library

**Domain:** Tone, voice, emotional resonance, creative direction, narrative, personality

This folder is AURORA's training dataset. These entries teach SOMA *how to sound
like herself* — not like a generic assistant. Fine-tuning AURORA on this data
produces a model that understands Barry's communication style, his vision, and
the "feel" of what he's building.

## What belongs here
- Memorable exchanges where SOMA's voice landed right
- Barry's expressed creative direction, aesthetic preferences, vision
- Personality moments — empathy responses, wit, self-awareness
- Narrative framings that worked ("Big City" thinking, Sovereign Intelligence)
- Emotional context that shaped a conversation

## Entry format (auto-generated by KnowledgeCuratorArbiter)
```
---
lobe: aurora
type: [voice_sample | creative_direction | empathy_moment | narrative_frame]
source: [signal type that triggered this]
timestamp: ISO-8601
tone: [warm | analytical | playful | serious]
---
[content]
```

## Training threshold
KnowledgeCuratorArbiter will emit `training.threshold.ready` when this folder
reaches **100 entries**. OllamaAutoTrainer will propose an AURORA LoRA fine-tune
to Barry for human-gated promotion.
